# 新手如何搭建自己的提示词素材库

提示词素材库不是把好句子堆在一起，而是把可复用的任务方法沉淀下来。它应该让你快速找到适合当前场景的模板，并知道该替换哪些变量。

## 建议字段

每条提示词至少保留这些字段：

| 字段     | 用途                       |
| -------- | -------------------------- |
| 场景     | 说明这条提示词适合什么任务 |
| 输入变量 | 标记每次需要替换的内容     |
| 输出格式 | 让结果可以被复制到后续流程 |
| 示例     | 给模型一个稳定参照         |
| 复盘     | 记录哪里有效、哪里需要改   |

## 分组方式

可以先按使用场景分组，例如写作、运营、学习、代码、分析。等数量变多后，再增加标签，例如标题、改写、检查、总结。

## 一个简单模板

```text
你是 {角色}。
请基于 {输入材料} 完成 {任务目标}。
要求：
- 语气：{语气}
- 长度：{长度}
- 禁止：{禁止事项}
输出格式：
{输出结构}
```

先保持模板少而精，等真实使用中出现重复需求，再扩充素材库。
